# 概述

1. App.vue 用于平时做组件开发临时预览
2. Storybook 用于协作开发共享，注意编写好 props 注释

## 模块

### 内容单页 single

主要是用于详情页的展示内容

- CmsSingle 单页框架
- Author 侧边栏作者信息
- PostDirectory 目录
- PostHeader 创作信息
- Creators 联合创作
- Collection 小册
- Article 文章内容
- Thx 交互组件
- SimpleThx 简单交互组件集合
- Comment 评论

### 筛选模块 list

主要用于各栏目的列表筛选

- clientBy 客户端筛选
- markBy 标记筛选
- menuBy 通用筛选
- orderBy 排序筛选
- tagBy 标签筛选
- topicBy 主题筛选
- versionBy 版本筛选
- zlpBy 资料片筛选

### 个人模块 author

一般用于文章详情的左侧边栏，个人主页等

-   AuthorFans 粉丝榜
-   AuthorFollow 关注用户（区分rss，此处用户更新不会收到消息）
-   AuthorGift 打赏用户
-   AuthorHonor 用户称号
-   AuthorInfo 用户基础信息
-   AuthorLink 用户绑定平台账号（微博，github，直播间）
-   AuthorMedals 用户勋章
-   AuthorPosts 用户作品
-   AuthorRss 订阅用户
-   AuthorTeams 所在团队
-   Avatar 头像
-   UserPop 搜索用户弹窗
